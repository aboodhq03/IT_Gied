# Feature F1: User Registration

## Description
This feature allows new users to create an account on the IT Guide platform by providing basic personal and academic information. It serves as the entry point for accessing all system functionalities.

## Implemented Functionality
- User registration using email, university ID, and password
- Input validation for required fields
- Secure password storage using hashing
- Storing user data in the database

## Related Requirements
- FR-1
- NFR-1

## Implementation Location
- `Controllers/UserController.cs`
