# Feature F2: User Login & Authentication

## Description
This feature enables registered users to securely log in to the system and access personalized content based on their role.

## Implemented Functionality
- Credential validation
- Secure authentication using ASP.NET Identity
- Redirect to dashboard upon successful login
- Error handling for invalid credentials

- NFR-2
- NFR-3
