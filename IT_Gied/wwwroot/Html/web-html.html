HTML من الصفر إلى الاحتراف

صفحة تدريبية كاملة: مسار تعلم مرتب، فيديوهات ومراجع رسمية، وتمارين + كويزات تفاعلية لتقييم فهمك.

المسار فيديوهات مراجع كويزات
Beginner — الانطلاقة

1) أساسيات HTML

  • هيكل الصفحة: <!doctype html> و <head> و <body>
  • العناصر الأساسية: headings, paragraphs, links, images
  • القوائم والجداول (Basics)
  • المسارات: Relative vs Absolute

تمرين: أنشئ صفحة CV بسيطة فيها صورة وروابط وقائمة مهارات.

Intermediate — بناء صفحات سليمة

2) HTML الدلالي + Forms

  • Semantic: header, nav, main, section, article, footer
  • Forms: input types, labels, validation attributes
  • Media: audio/video + iframe
  • SEO basics: title, meta description

تمرين: صفحة تسجيل (Register) مع labels والتحقق (required, pattern, minlength).

Advanced — احتراف

3) Accessibility + Best Practices

  • Accessibility: alt, labels, tabindex
  • ARIA basics: متى تستخدمها ومتى تتجنبها
  • Performance: صور مناسبة + lazy loading
  • Clean markup: تجنب div soup واعتمد الدلالات

تمرين: حوّل صفحة غير دلالية إلى Semantic مع تحسين الوصول.

خطة تطبيق (مشاريع صغيرة)

  1. Landing Page: هيدر + أقسام + CTA + فوتر.
  2. Blog Layout: article + aside + pagination.
  3. Form Page: تسجيل + HTML validation.
  4. Portfolio: مشاريع + روابط + أقسام دلالية.

قاعدة ذهبية: قبل CSS، تأكد أن الـ HTML وحده منطقي ومقروء (Semantic).
