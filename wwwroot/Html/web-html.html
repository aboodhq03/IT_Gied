HTML From Beginner to Professional

A complete training page: an organized learning path, videos and official references, plus exercises and interactive quizzes to evaluate your understanding.

Learning Path Videos References Quizzes
Beginner — Getting Started

1) HTML Basics

  • Page structure: <!doctype html>, <head>, and <body>
  • Basic elements: headings, paragraphs, links, images
  • Lists and tables (Basics)
  • Paths: Relative vs Absolute

Exercise: Create a simple CV page containing an image, links, and a skills list.

Intermediate — Building Proper Pages

2) Semantic HTML + Forms

  • Semantic: header, nav, main, section, article, footer
  • Forms: input types, labels, validation attributes
  • Media: audio/video + iframe
  • SEO basics: title, meta description

Exercise: Build a Register page with labels and validation (required, pattern, minlength).

Advanced — Professional Level

3) Accessibility + Best Practices

  • Accessibility: alt, labels, tabindex
  • ARIA basics: when to use it and when to avoid it
  • Performance: optimized images + lazy loading
  • Clean markup: avoid div soup and use semantic elements

Exercise: Convert a non-semantic page into a semantic one with better accessibility.

Practice Plan (Mini Projects)

  1. Landing Page: Header + sections + CTA + footer.
  2. Blog Layout: article + aside + pagination.
  3. Form Page: registration + HTML validation.
  4. Portfolio: projects + links + semantic sections.

Golden rule: Before CSS, make sure your HTML alone is logical and semantic.
